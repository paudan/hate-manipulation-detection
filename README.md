# Hate and manipulation detection using transformers

A transformer-based classifier that can be used to train models for hate or emotional manipulation detection

This is a research code for AICP-FIMI project: _AI-driven cloud platform to counter FIMI (Foreign Influence and Manipulation of Information) during elections 
and early warning service for identification of social media bots and troll farms (AICP-FIMI)_
